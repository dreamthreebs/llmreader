# LLM Paper Reader

用 Cursor Agent 精读物理/宇宙学论文的工作区。采用「双层阅读模式」：先忠实对齐原文，再解释物理图景。

## 快速开始

### 1. 放入论文

从 arXiv 下载 LaTeX 源码，放入 `papers/` 目录：

```
papers/
└── 1511.02846/
    ├── ms.tex
    ├── defs.tex
    └── figures/
```

或者直接给 Agent arXiv 链接，它会自动下载。

### 2. 口令表

| 口令 | 作用 | 示例 |
|------|------|------|
| **读 §X** | 按 subsection 精读，一次一个 subsection | `读 §2` 从 §2.1 开始；`读 §2.3` 直接读 §2.3 |
| **读公式 (X)** | 精读某个公式，展开符号表 + 逐项解释 | `读公式 (3)` |
| **鸟瞰 §X** | 快速概览某节的结构、假设、结论 | `鸟瞰 §3` |
| **串联公式** | 把已读过的关键公式串联，梳理推导链 | `串联公式 §2` |
| **回顾** | 回顾最近读过的几段，总结逻辑推进线 | `回顾` |
| **追问** | 对刚才读的内容追问（某项含义、近似影响等） | `公式 (5) 右边第二项物理上是什么？` |
| **批判** | 审视关键假设和近似，找跳步 | `批判 §2.3 的假设` |
| **下一段** | 继续读当前 section 的下一段 | `下一段` |
| **检查** / **校验** | 核对笔记准确性（见下方详细说明） | `检查 §1` |

### 3. 推荐阅读流程

```
第一遍  →  鸟瞰全文（给出 section 标题列表即可）
第二遍  →  读 §2（核心方法和公式）
第三遍  →  读 §3（数值方法和结果）
第四遍  →  串联公式，梳理推导链
第五遍  →  批判，审视假设和近似
```

### 4. 输出格式

每次精读输出 7 个固定栏目：

1. **Original Text** — 原文引用
2. **Literal Translation** — 逐句直译
3. **Faithful Paraphrase** — 通俗重述（不改原意）
4. **Logical Role** — 每句的逻辑角色（定义/假设/近似/推理/结论…）
5. **Physics Meaning** — 物理图景、对象、过程、控制参数
6. **Equation / Symbol Notes** — 公式符号表、逐项解释、数学结构
7. **Consistency Check** — 区分原文/重述/解释/模糊之处

每句解释标注来源：**[原文]** / **[重述]** / **[补充]**

### 5. 校验功能

读完一个 section 后，可以用 `检查 §X` 让 Agent 回到 LaTeX 源文件逐项核对笔记的准确性。校验分五步：

| 步骤 | 检查内容 |
|------|----------|
| Step 1：原文核对 | 笔记引用是否与 `.tex` 源文件一致，有无遗漏或抄错 |
| Step 2：翻译校验 | 翻译是否忠实，有无遗漏限定词或改变语气 |
| Step 3：物理内容校验 | 物理图景、因果关系、近似条件是否正确 |
| Step 4：来源标注校验 | [原文]/[补充] 标记是否准确 |
| Step 5：汇总 | 整体评估，列出所有问题 |

每项用 ✅ / ⚠️ / ❌ 标记，结果追加到笔记文件的「校验记录」section。

## 目录结构

```
llmreader/
├── papers/          # 原始文献（git 忽略，不上传）
├── readings/        # 阅读笔记（一篇论文一个 .md）
├── index.md         # 文献索引
├── README.md        # 本文件
└── .cursor/
    ├── rules/
    │   └── paper-reader.mdc      # Agent 行为规则
    └── skills/
        ├── paper-reader/
        │   ├── SKILL.md           # 双层阅读模板
        │   ├── examples.md        # 输出示例
        │   └── workflow.md        # 工作流细节
        └── paper-checker/
            └── SKILL.md           # 校验笔记准确性
```

## 注意事项

- 优先给 LaTeX 源码（`.tex`），PDF 效果会打折
- 大多数 arXiv 论文可在页面点 "Other formats → Download source" 获取源码
- `papers/` 已被 `.gitignore` 排除，不会提交到 git
