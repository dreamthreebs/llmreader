# 论文阅读校验 (Paper Checker)

对 `readings/<id>.md` 中的阅读笔记进行逐项校验，确保原文引用准确、翻译忠实、物理解释无误。

## 触发口令

`检查` / `校验` / `check`

## 校验流程

收到校验请求后，按以下步骤执行：

### Step 1：原文核对

逐段将笔记中的 Original Text 与 `papers/<id>/` 中的 LaTeX 源文件比对：

- 是否有遗漏的句子或半句
- 是否有抄写错误（拼写、公式符号）
- 摘要概括的段落（如 [¶3–5 历史回顾，摘要]）是否遗漏了关键信息
- 公式是否与源文件一致

输出格式：

```
✅ ¶1 — 原文正确
⚠️ ¶3 — 遗漏：原文还提到 XXX
❌ ¶7 — 公式 (X) 符号抄错：原文是 A，笔记写成 B
```

### Step 2：翻译校验

检查 Literal Translation 是否忠实：

- 是否改变了原文意思
- 是否遗漏了限定词（approximately, assume, neglect, leading order 等）
- 是否把不确定的表述翻译成了确定性陈述
- 术语翻译是否准确

输出格式：

```
✅ ¶1 译 — 忠实
⚠️ ¶6 译 — "sensitive to" 翻译为"敏感"没问题，但原文 "mainly because" 的"主要"被省略
❌ ¶9 译 — 原文说 "suggests"（暗示），翻译成了"证明"，语气过强
```

### Step 3：物理内容校验

检查 Faithful Paraphrase、Physics Meaning、追问解答中的物理内容：

- 物理图景描述是否正确
- 因果关系是否颠倒
- 近似条件是否遗漏
- 量级估计是否合理
- 是否把 Agent 补充的内容误标为 [原文]，或反过来
- 公式符号含义是否与原文定义一致

输出格式：

```
✅ Physics Meaning — 物理图景正确
⚠️ ¶1 追问 — "势涨落决定物质分布"的因果链正确，但严格来说势涨落和密度涨落通过 Poisson 方程互相决定，不是单向的
❌ Equation Notes — 符号 X 的含义写错，原文定义的是 Y
```

### Step 4：来源标注校验

检查 [原文] / [重述] / [补充] 标注是否正确：

- 标了 [原文] 的内容是否真的在论文中明确写出
- 标了 [补充] 的内容是否确实不在原文中
- 有没有漏标的补充内容（看起来像原文说的但其实是 Agent 推断）

### Step 5：汇总

最后给出整体评估：

```
## 校验结果

- 原文引用：X/Y 段正确，Z 处有问题
- 翻译忠实度：整体良好 / 有 N 处需修正
- 物理内容：正确 / 有 N 处需注意
- 来源标注：准确 / 有 N 处需修正
```

## 行为约束

1. 必须回到 LaTeX 源文件核对，不能凭记忆判断。
2. 对物理内容的判断要保守——如果不确定某个物理说法是否正确，标为 ⚠️ 而非 ❌，并说明存疑原因。
3. 不要在校验过程中大幅重写笔记——只指出问题，由用户决定是否修改。
4. 校验结果写入笔记文件末尾的「校验记录」section。
